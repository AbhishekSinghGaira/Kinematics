Kinematics Formula Cheat Sheet

1. Basic Kinematic Equations

Formula Description
s = ut + ½at² Displacement with initial velocity \(u\) and acceleration \(a\)
v = u + at Final velocity after time \(t\)
v² = u² + 2as Velocity-Displacement relation
sₙ = u + ½a(2n-1) Displacement in the \(n^{th}\) second
v_avg = (u + v) / 2 Average velocity

2. Relative Motion

Formula Description
v_AB = v_A - v_B Relative velocity of A with respect to B
Δx_rel = (v_A - v_B)t Relative displacement over time \(t\)

3. Free Fall (Vertical Motion)

Formula Description
h = ut + ½gt² Displacement under gravity (\(g\))
v = u + gt Final velocity under gravity
v² = u² + 2gh Velocity-displacement relation for free fall
t = u/g Time to reach maximum height
H = u²/2g Maximum height reached
T = 2u/g Total time of flight

4. Projectile Motion

Formula Description
R = (u² sin(2θ)) / g Horizontal range
H = (u² sin²θ) / 2g Maximum height
T = (2u sinθ) / g Time of flight
x = u cosθ × t Horizontal displacement
y = u sinθ × t - ½gt² Vertical displacement
y = x tanθ - (g x²) / (2u² cos²θ) Equation of trajectory

5. Circular Motion

Formula Description
a_c = v² / r Centripetal acceleration
F_c = mv² / r Centripetal force
ω = v / r Angular velocity
v = rω Relation between linear and angular velocity

6. Miscellaneous

Formula Description
t = √(2h / g) Time to reach the ground in free fall
v_avg = Total Distance / Total Time Average speed for non-uniform motion
v_rain, man = √(v_r² + v_m²) Relative velocity in rain problems

Tips for IIT-JEE Preparation

  • Draw clear diagrams for every problem.
  • Break motion into components (horizontal and vertical).
  • Write down given values and choose appropriate equations.
  • Focus on common scenarios: projectile motion, inclined planes, and relative velocity.
